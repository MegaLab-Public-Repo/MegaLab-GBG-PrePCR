SPT Labtech Dragonfly

Driver User Guide

Version 3.2.7523.28495

Introduction:

This driver provides an interface to the SPT Labtech Dragonfly. This document details the installation and setup instructions necessary to use the driver.

Installing the Driver:

For automated installation, use the 'Update' feature from within Green Button Go in the 'About' window.

To manually install the driver, copy the following file(s) to the 'C:\Program Files (x86)\Green Button Go\Drivers' folder:

SPTLabtech.Dragonfly.Driver.dll
DragonflyAutomation.dll
DragonflySharedTypes.dll

Omit the '(x86)' in the destination folder on 32-bit operating systems. If the file was downloaded from the Internet, it may be necessary to unblock the file. First copy it to a location where you have write privileges (such as the Desktop or My Documents folder), then right click on the dll, select Properties and click or uncheck the 'Unblock' button if it exists.

Connection:

The Dragonfly must be connected to the computer running Green Button Go (or the Green Button Go Remote Client).

Required Hardware:

SPT Labtech Dragonfly.

Required Software:

SPT Labtech Dragonfly Discovery software.

Instrument Properties:

Processes:

  1. Run All Layers
    • Default: Run the protocol for all layers.
      • Parameters
        • Experiment File - The file containing the protocol to run.
  2. Run Layer
    • Default: Run the protocol for a single layer.
      • Parameters
        • Experiment Layer - The zero-based index of the layer.
    • Load Experiment: Run the protocol for a single layer from the specified experiment file.
      • Parameters
        • Experiment File - The file containing the protocol to run.
        • Experiment Layer - The zero-based index of the layer.

Commands:

  1. Load Experiment File: Load the experiment file.
    • Input Parameters
      • Experiment File - The file containing the protocol to run.
  2. Run Current Layer: Run the currently selected layer of the experiment file.
  3. Run All Layers: Run all the layers of the experiment file.
  4. Stop Run: Stops the current run.
  5. Reset: Resets the Dragonfly instrument.
  6. Park and Release: Parks and releases the nest.
  7. Park Under Lid: Parks the nest under the lid.
  8. Switch to Next Layer: Switches to the next experiment file layer.
  9. Switch to Previous Layer: Switches to the previous experiment file layer.
  10. Switch Layer Index: Switches to the specified experiment file layer.
    • Input Parameters
      • Experiment Layer - The index of the layer to switch to.
  11. Run Simulation: Simulate a run on the instrument.
  12. Get Status: Get the currently selected layer for the current experiment file.
    • Output Parameters
      • Status - Last updated status message.
      • Experiment Folder - Returns the folder path for the default location where Dragonfly experiment and CSV files are stored.
      • Total Layers - Returns the total number of layers in the currently loaded experiment
      • Current Layer - Returns the 0-based index of the currently selected layer
      • Parked And Released - True if the deck is in the parked position with the clamp released.
      • Parked Under Lid - True if the deck is parked under the incubation lid
  13. Stop Simulation: Stops any current simulation dispense run.
  14. Load CSV File: Attempts to load a Dragonfly platemap(.csv) file.
    • Input Parameters
      • File Path - The .CSV file path.
  15. Get Syringe Volumes: Returns a list of volumes, in μl, in each syringe. List ordered in accordance with the head index.
    • Output Parameters
      • Syringe Volumes (ul) - List of volumes, in μl, in each syringe
  16. Get Syringes Fitted: Returns a list of True/False values that indicate which heads have syringes connected. List is ordered in accordance with head indexes.
    • Output Parameters
      • Syringes Fitted - List of True/False values that indicate which heads have syringes connected.
  17. Multi Aspirate: Aspirates for the currently selected layer the specified number of times.
    • Input Parameters
      • Multiple - Aspirate multiple.
  18. Aspirate Current Layer: Aspirates all syringes with assigned heads in the current layer.
  19. Aspirate To Volume: Aspirate the specified volume of liquid, in μl. The volume is specified by a comma-separated list of syringe index and volume values (in ul) that are in turn separated by colons. The format is '<index 1>:<volume 1>, <index 2>:<volume 2>, <index 3>:<volume 3>,...'
    • Input Parameters
      • Volumes - A comma-separated list of syringe index and volume values (in ul) that are in turn separated by colons. The format is '<index 1>:<volume 1>, <index 2>:<volume 2>, <index 3>:<volume 3>,...'
  20. Aspirate To Max Volume: Aspirates syringes to their maximum volume, identified by their head index. Syringes whose head indexes do not appear in the list will not aspirate.
    • Input Parameters
      • Head Indexes - A comma-separated list of syringe head indexes.
  21. Purge All: Purges all heads that have a syringe fitted.
  22. Purge And Remove All: Purges and removes all heads that have a syringe fitted.
  23. Purge And Remove: Purges and removes all heads in the list that have a syringe fitted.
    • Input Parameters
      • Head Indexes - A comma-separated list of syringe head index.
