Liquid Handler Driver - By Request

Introduction

The By Request process allows the liquid handler to request the loading and unloading of labware. This is done through a handshake file. It is assumed that the liquid handler method is running during this process and will use its own internal scripting or file writing capabilities to write and read from the handoff file.

Setup

The By Request process is inserted into the plate process where desired. The Status File parameter needs to be set to the desired file that will be used for handshaking, and the nest position needs to be specified.

Operation

To trigger a plate to be loaded into this process the liquid handler needs to write the following text to the file:

load

Once GBG has loaded the labware onto the specified nest, GBG will overwrite the file with the following text:

loaded

The liquid handler needs to monitor for this change and can then proceed with processing the labware.

To remove labware from the nest, the liquid handler needs to write the following text to the file:

remove

Once GBG had remove the labware from the nest it will overwrite the file with the following text:

removed
